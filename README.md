![中国方言](https://img.shields.io/badge/中国方言-传承与整理-blue.svg) ![License](https://img.shields.io/badge/License-MIT-green.svg) ![Contributors](https://img.shields.io/badge/Contributors-Welcome-red.svg)

📖 仓库介绍 (Repository Introduction)

本仓库致力于 中国方言的收集、整理、保存与传播，旨在搭建一个开源、共建的方言资源平台，助力方言文化的传承与普及。

中国方言是中华优秀传统文化的重要组成部分，承载着地域历史、民俗风情和民族记忆。随着城市化进程加快，许多方言面临弱化、流失的困境，本仓库希望通过技术手段，系统性整理各地方言的语音、词汇、语法、俗语、民谣等内容，为方言研究、学习和保护提供便捷的资源支持，让更多人了解、关注并参与到方言传承中来。

无论是方言爱好者、语言研究者，还是普通使用者，都可以在这里找到有用的方言资源，也可以参与贡献，共同丰富这份属于中国人的文化财富。

🗺️ 方言覆盖范围 (Dialect Coverage)

本仓库计划逐步覆盖中国主要方言区，目前已整理/待整理的方言包括（按方言区分类）：

- 北方方言区：北京话、东北话、河南话、陕西话、山东话等

- 吴方言区：上海话、苏州话、杭州话、宁波话等

- 湘方言区：长沙话、湘潭话、衡阳话等

- 赣方言区：南昌话、赣州话、吉安话等

- 客家方言区：梅州话、惠州话、龙岩话等

- 闽方言区：福州话、厦门话（闽南语）、泉州话、莆田话等

- 粤方言区：广州话、深圳话、佛山话、潮汕话等

- 其他小众方言：温州话、徽州话、儋州话等

欢迎补充未列出的方言，共同完善方言覆盖体系。

🔍 资源说明 (Resource Description)

本仓库的方言资源主要包含以下类型，所有资源均力求准确、实用：

1. 语音素材：包含方言日常对话、俗语、民谣等音频文件，标注发音人、地域（精确到市县），方便大家聆听真实的方言发音。

2. 词汇对照：整理各地方言常用词汇，提供“方言词汇-普通话释义-拼音标注”对照，涵盖日常用品、饮食、情感、动作等场景。

3. 语法要点：简要说明各方言的语法特点（与普通话的差异），包含句式结构、虚词用法、语气词等。

4. 俗语与文化：收集各方言的俗语、谚语、歇后语、民谣、民间故事，结合地域文化背景进行解读，展现方言背后的文化内涵。

5. 研究资料：整理方言相关的学术研究文章、历史文献、保护政策等，为方言研究提供参考。

🤝 如何贡献 (How to Contribute)

方言的传承需要大家的共同努力，欢迎每一位爱好者参与到本仓库的建设中来！无论是补充资源、修正错误，还是分享方言故事，都是宝贵的贡献。

贡献步骤

1. Fork 本仓库到自己的GitHub账号；

2. 克隆（Clone）自己Fork的仓库到本地；

3. 按照仓库结构，补充/修改相关资源（注意：补充内容需准确，标注来源和地域）；

4. 提交（Commit）修改，填写清晰的提交说明（如“补充上海话常用词汇”）；

5. 推送（Push）到自己的Fork仓库；

6. 提交Pull Request，说明贡献内容，等待审核合并。

贡献规范

- 内容准确：方言词汇、发音、语法等内容需真实可靠，尽量标注来源（如本地人口述、权威文献）；

- 格式统一：按照仓库现有格式整理内容，避免混乱（如词汇对照统一使用“方言-普通话-拼音”格式）；

- 尊重差异：各方言存在地域变体（如同一方言不同市县的差异），请标注具体地域，不随意否定不同变体；

- 拒绝侵权：不提交侵权内容（如未经授权的音频、文献），所有贡献内容需符合开源协议。

详细贡献规范请参考 CONTRIBUTING.md。
